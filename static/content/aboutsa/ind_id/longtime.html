Anda tidak menggunakan bahasa untuk sekian waktu lamanya

Jika sudah lama sekali Anda terakhir belajar atau menggunakan bahasa tersebut, dan Anda tidak punya kesempatan untuk menggunakannya baru-baru ini, mungkin sulit untuk memperkirakan seberapa baik kemampuan Anda saat ini.

Kemahiran bahasa biasanya memburuk dari waktu ke waktu jika tidak digunakan, tetapi tingkat lupanya bervariasi. Sebagai contoh, hal ini biasanya terjadi bahwa kemampuan Anda untuk berbicara dan menulis lebih buruk daripada pemahaman Anda.
