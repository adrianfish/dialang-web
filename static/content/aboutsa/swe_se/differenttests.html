Hur man jämför olika test

Om du vill jämföra vad olika testresultat och testnivåer innebär och bedöma om de kan vara till nytta för dig som språkinlärare bör du åtminstone ta ställning till nedanstående punkter. Ju mer testen skiljer sig åt desto mindre sannolikt är det att resultaten ger dig samma information om din språkfärdighet.

Syftet med testet: Ett test som används för att bedöma din nivå vid slutet av en kurs ger antagligen inte samma information som ett test som är ämnat att mäta dina färdigheter i absoluta termer enligt en internationell skala med information om vad inlärare på en viss nivå kan klara i verkliga livet.

Testets innehåll: Ett test som mäter ett särskilt ämnesområde eller har ett visst yrkesinriktat innehåll ger inte samma information som ett test som täcker flera olika områden och 'allmän språkfärdighet'.

Testets bredd: Vissa test omfattar flera olika färdighetsvariabler medan andra endast är inriktade på skriftlig eller muntlig färdighet. Ta reda på om färdigheterna viktas på olika sätt i de test som du jämför. Jämför alltid sådana delar som låter sig jämföras: den övergripande/sammantagna nivån med en annan övergripande/sammantagen nivå, eller nivån i muntlig färdighet med en annan nivå i muntlig färdighet i ett annat test.

Testmetod: I en del test finns flera olika uppgiftsstyper (flervalsuppgifter, korta svar, luckor, matchningsuppgifter etc.) medan det i andra test kanske endast finns en eller två uppgiftstyper. Sådana skillnader kan medföra olika resultat.
