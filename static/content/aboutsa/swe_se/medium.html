Mediet för testningen

En del test ges via ett medium som eventuellt är obekant eller till och med avskräckande för vissa språkinlärare. Som exempel kan nämnas test som utförs i inlärningsstudio (språklaboratorium) eller med ordbehandlare. Om man är ovan vid datorer kan ett test som genomförs med dator påverka resultatet. Exempelvis läser de flesta en text långsammare på bildskärm än på papper.
