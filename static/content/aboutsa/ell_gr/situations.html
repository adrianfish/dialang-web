Χρησιμοποιείτε τη γλώσσα σε διαφορετικές καταστάσεις
  • Χρησιμοποιείτε τη γλώσσα μόνο για ειδικούς σκοπούς ή σε ειδικές συνθήκες -για παράδειγμα σε σχέση με τη δουλειά σας, την κοινωνική σας ζωή, για το αντικείμενο της μελέτης σας ή για προσωπικό σας ενδιαφέρον. Γι' αυτό γνωρίζετε καλά τη θεματική περιοχή και μπορεί να είστε σε θέση να μαντέψετε άγνωστες λέξεις και εκφράσεις από τα συμφραζόμενα. Το DIALANG, όμως, καλύπτει μεγάλη ποικιλία κειμένων και περιοχών και γι' αυτό το λόγο οι γλωσσικές σας γνώσεις σας μπορούν 'να επεκταθούν'.
  • Η επιτυχής επικοινωνία σας σε περιορισμένη περιοχή γλωσσικών συνθηκών μπορεί να σας έχουν οδηγήσει στην υπόθεση ότι θα μπορούσατε να λειτουργήσετε με τον ίδιο βαθμό επιτυχίας σε ευρύτερη περιοχή γλωσσικών συνθηκών.
  • Εξάλλου η ικανότητά σας να χρησιμοποιείτε τη γλώσσα σε διαφορετικές θεματικές περιοχές και καταστάσεις μπορεί να αναπτυχθεί πολύ περισσότερο απ' όσο νομίζατε. Αυτό μπορεί να έχει σχέση με άλλους παράγοντες, αλλά θα μπορούσε επίσης να οφείλεται στο ότι οι καταστάσεις που χρησιμοποιείτε τη γλώσσα (π.χ. για γενικούς κοινωνικούς σκοπούς) σας βοηθούν πολύ στο να αναπτύξετε σημαντικές μορφές όπως το λεξιλόγιο και έτσι να μπορείτε να λειτουργήσετε σε ευρύτερη κλίμακα γλωσσικών συνθηκών.
