Χρησιμοποιείτε τη γλώσσα σπάνια

Πόσο συχνά χρησιμοποιείτε τη γλώσσα που μαθαίνετε ώστε να επενεργήσει στις γλωσσικές σας δεξιότητες.

Όσο περισσότερες ευκαιρίες είχατε να χρησιμοποιήσετε τη γλώσσα σε διαφορετικές καταστάσεις και για διαφορετικούς σκοπούς τόσο περισσότερες είναι οι πιθανότητες να έχετε αναπτύξει μια αρκετά σωστή εικόνα του τι μπορείτε και του τι δεν μπορείτε να κάνετε κατά τη χρήση της γλώσσας. Όσο λιγότερες ευκαιρίες είχατε τόσο πιο δύσκολο είναι να σχηματίσετε μια ιδέα του πόσο πραγματικά καλός/ή είστε.
