Υπερεκτιμάτε τις ικανότητές σας, γιατί είστε καλοί/ες σε κάποιες δεξιότητες

Είναι πιθανό να μπορείτε να διαβάζετε μία ποικιλία διαφορετικών κειμένων σε κάποια άλλη γλώσσα χωρίς μεγάλη δυσκολία (ειδικά αν αυτά τα κείμενα είναι για θέματα που τα γνωρίζετε). Όμως, μπορεί να βρείτε προβληματική την παραγωγή γραπτού λόγου, ιδιαίτερα αν πρέπει να γράψετε αρκετά επίσημα κείμενα που απαιτούν υψηλού επιπέδου ακρίβεια και συνοχή. Ή μπορεί να διαπιστώσετε ότι ούτε η παραγωγή και η κατανόηση προφορικού λόγου είναι τόσο εύκολες, γιατί πρέπει να αντιμετωπίσετε, για παράδειγμα, διάφορους θορύβους, ομιλία σε ταχύ ρυθμό και μη γνώριμη προφορά.

Με τον ίδιο τρόπο, μπορεί να έχετε βρει την παραγωγή προφορικού λόγου αρκετά εύκολη και να θεωρείτε ότι μπορείτε να γίνετε κατανοητός/ή στις περισσότερες περιπτώσεις, ιδιαίτερα σ' εκείνες που σας είναι οικείες. Όμως, μπορεί να βρείτε την παραγωγή γραπτού λόγου προβληματική (ιδιαίτερα την πιο επίσημη μορφή της), γιατί πολλές περιπτώσεις απαιτούν υψηλότερο επίπεδο ακρίβειας απ' ό,τι οι παρόμοιες περιπτώσεις παραγωγής προφορικού λόγου. Και μπορεί να διαπιστώσετε ότι κείμενα και λόγος που περιέχουν πολλές συμπυκνωμένες, αφηρημένες πληροφορίες και ιδέες ίσως είναι δύσκολα κατανοητά, γιατί κατά την ομιλία σας μπορεί να είστε σε θέση να επιλέξετε πότε, για τι και πώς να μιλήσετε και έτσι να αποφύγετε να χρησιμοποιήσετε τις εκφράσεις εκείνες που δε σας είναι οικείες.
