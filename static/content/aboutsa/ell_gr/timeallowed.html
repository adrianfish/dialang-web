Επιτρεπόμενος χρόνος

Στις περισσότερες εξετάσεις έχετε μόνον ένα περιορισμένο χρονικό διάστημα στη διάθεσή σας για να ολοκληρώσετε τα θέματα, ενώ στην πραγματική ζωή μπορεί να έχετε περισσότερο διαθέσιμο χρόνο, όπως, για παράδειγμα, όταν διαβάζετε ένα κείμενο. Αλλά μπορεί να συμβαίνει και το αντίθετο: μπορεί να βρείτε μερικές ρεαλιστικές καταστάσεις περισσότερο δύσκολες από αυτές των εξετάσεων, γιατί πρέπει να κατανοήσετε και να αντιδράσετε πιο γρήγορα απ' ότι σε συνθήκες εξέτασης.

Δε διαφέρει μόνο ο χρόνος που έχετε στη διάθεσή σας για να ολοκληρώσετε τα θέματα από την πραγματική ζωή, αλλά μπορεί επίσης να διαφέρει και ο επιτρεπόμενος χρόνος για να προετοιμαστείτεε για τις ερωτήσεις. Στην πραγματική ζωή συχνά μπορεί να έχετε πολύ χρόνο να προετοιμαστείτε για μια παρουσίαση, για παράδειγμα, ενώ ο χρόνος προετοιμασίας κατά τη διάρκεια εξέτασης ίσως είναι πολύ περιορισμένος και αυτό μπορεί να μειώσει την απόδοσή σας.

Ο διαθέσιμος χρόνος επηρεάζεται επίσης από την ευκαιρία να ελέγξετε και να διορθώσετε ότι έχετε γράψει. Στην πραγματική ζωή μπορεί να χρησιμοποιείτε έναν επεξεργαστή και να έχετε ώρα για να διορθώσετε ότι έχετε γράψει. Στις περισσότερες εξετάσεις δεν υπάρχει χρόνος να ξαναγράψετε τα κείμενά σας.
