Επιδιωκόμενο επίπεδο επάρκειας

Το πόσο ψηλό είναι το επιδιωκόμενο επίπεδο για την εκμάθηση της γλώσσας μπορεί να επηρεάσει την εκτίμηση της επάρκειάς σας. Για παράδειγμα, αν είστε αρχάριος ή μέσος σπουδαστής και έχετε ως στόχο να αποκτήσετε την επάρκεια του φυσικού ομιλητή, θα έχετε ένα μακρύ δρόμο να κάνετε και ίσως νιώθετε ότι προχωρείτε πολύ αργά και ότι πιθανόν δεν μπορείτε να είστε πολύ καλός/ή -συνεπώς πιθανόν να υποτιμήσετε τον εαυτό σας.

Αν έχετε θέσει περιορισμένους στόχους, πιθανότατα να τους πετύχετε σε λογικό χρονικό διάστημα, μερικές φορές μάλιστα πολύ γρήγορα. Το πρόβλημα που μπορεί αυτό να παρουσιάσει στην αυτοαξιολόγηση είναι ότι μπορεί να μην έχετε συνειδητοποιήσει πόσο περιορισμένος ήταν ο στόχος και ότι άλλες περιοχές της γλώσσας μπορεί να είναι πιο ενδιαφέρουσες. Σ' αυτή την περίπτωση, πιθανότατα να υπερεκτιμήσετε την επάρκειά σας.

Ισχύον επίπεδο επάρκειας

Σουδαστές σε διαφορετικά επίπεδα μπορεί να εκτιμούν το επίπεδο της γλωσσικής τους επάρκειας με διαφορετικό τρόπο. Οι αρχάριοι μπορεί να είναι τόσο ευχαριστημένοι με τη γρήγορη αρχική τους πρόοδο ώστε να υπερεκτιμούν τις ικανότητές τους. Αλλά είναι επίσης δυνατό να αισθάνονται ότι έχουν τόσα πολλά να μάθουν ακόμη ώστε να υποτιμούν την επάρκειά τους. Μερικοί μέσοι σπουδαστές μπορεί να κατέχουν τη γλώσσα αρκετά καλά για τους δικούς τους ειδικούς σκοπούς, αλλά να μην αντιλαμβάνονται ότι η επάρκειά τους δεν είναι ικανοποιητική για άλλους σκοπούς.

Και οι μέσοι και οι προχωρημένοι σπουδαστές μπορεί να υποτιμούν την επάρκειά τους. Τυπικά έχουν κάνει γλωσσικά μαθήματα για αρκετά χρόνια, και μολονότι έχουν αποκτήσει υψηλό επίπεδο επάρκειας, γνωρίζουν ότι έχουν ακόμη πολλά να μάθουν, αν έχουν θέσει υψηλούς στόχους και επιθυμούν να πετύχουν ένα 'ιδανικό' επίπεδο επάρκειας, όπως αυτό πολλών φυσικών ομιλητών.
