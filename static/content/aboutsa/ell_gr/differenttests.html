Πώς να συγκρίνετε διαφορετικές εξεταστικές δοκιμασίες.

Όταν χρειάζεται να συγκρίνετε τη σημασία των αποτελεσμάτων και των βαθμών διαφορετικών εξεταστικών δοκιμασιών και να αξιολογήσετε την ενδεχόμενη χρησιμότητα τους για σας ως σπουδαστή της γλώσσας, πρέπει να ελέγξετε τουλάχιστο τα παρακάτω στοιχεία. Όσο περισσότερο διαφέρουν οι εξεταστικές δοκιμασίες μεταξύ τους, τόσο λιγότερο πιθανό είναι τα αποτελέσματά τους να σας δώσουν τις ίδιες πληροφορίες για τις γλωσσικές σας δεξιότητες.

Σκοπός της δοκιμασίας: μια δοκιμασία που χρησιμοποιείται για να σας δώσει ένα βαθμό στο τέλος ενός προγράμματος προφανώς δε σας δίνει τις ίδιες πληροφορίες που δίνει μια δοκιμασία με σκοπό να μετρήσει τις δεξιότητές σας με απόλυτους όρους σε μια διεθνή κλίμακα, η οποία σας λέει τι πρέπει να μπορούν να κάνουν σ' αυτό το επίπεδο οι σπουδαστές στην πραγματική ζωή.

Περιεχόμενο της δοκιμασίας: μια δοκιμασία που μετρά ένα συγκεκριμένο θέμα ή το περιεχόμενο επαγγελματικού τομέα δε θα σας δώσει τις ίδιες πληροφορίες με αυτές που δίνει μια δοκιμασία που καλύπτει μια κλίμακα διαφορετικών περιοχών και 'γενικής' γλωσσικής επάρκειας.

Πεδίο που καλύπτει η δοκιμασία: μερικές δοκιμασίες καλύπτουν ένα μεγάλο πεδίο δεξιοτήτων και απόψεων επάρκειας, ενώ άλλα εστιάζονται μόνο σε δεξιότητες γραπτού ή προφορικού λόγου. Ελέγξτε αν η βαρύτητα των δεξιοτήτων διαφέρει στις δοκιμασίες που συγκρίνετε. Πάντα συγκρίνετε ομοειδή: το συνολικό βαθμό με κάποιον άλλο συνολικό βαθμό, ή ένα βαθμό σε παραγωγή προφορικού λόγου με έναν άλλο βαθμό παραγωγής προφορικού λόγου.

Μέθοδοι που χρησιμοοιούνται στη δοκιμασία: μερικές δοκιμασίες χρησιμοποιούν μία ποικιλία ερωτήσεων (πολλαπλή επιλογή, σύντομες απαντήσεις, συμπλήρωση κενών, αντιστοίχιση κτλ.), ενώ άλλες δοκιμασίες μπορεί να χρησιμοποιούν ένα ή δύο τύπους ερωτήσεων. Αυτές οι διαφορές μπορεί να δώσουν διαφορετικά αποτελέσματα.
