Temps imparti

Dans la plupart des tests, vous disposez seulement d'un temps limité pour faire les exercices, tandis que dans la vie réelle, vous aurez éventuellement plus de temps, notamment quand vous lisez un texte par exemple. Mais le contraire peut aussi être vrai : dans la vie réelle, vous pourrez être confronté(e) à des situations plus difficiles que dans les tests, car vous aurez parfois besoin de comprendre et de réagir plus rapidement.

Non seulement le temps imparti pour faire le test peut différer de la vie réelle, mais aussi le temps dont vous disposez pour préparer les exercices. Dans la vie réelle, vous aurez souvent beaucoup de temps pour préparer une présentation par exemple, tandis que le temps de préparation dans un test peut être très limité, ce qui peut nuire à votre performance.

Le temps imparti a aussi une influence sur la relecture de ce que vous avez écrit. Dans la vie réelle, vous utilisez éventuellement un traitement de texte et vous avez le temps de relire ce que vous avez écrit. Dans la plupart des tests, vous ne pouvez pas recopier vos textes.
