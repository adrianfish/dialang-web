D'autres tests de langues vous donnent une appréciation différente de vos compétences

Les tests que vous faites à l'école, sur votre lieu de travail ou ailleurs, peuvent vous donner une idée de votre niveau de langue, mais il est important de tenir compte du fait que les résultats de différents tests peuvent signifier des choses très différentes. Ainsi, si vous voulez utiliser les tests de langue pour vous guider dans votre apprentissage et votre auto-évaluation, vous devez savoir un certain nombre de choses sur ces tests. Vous trouverez plus loin d'autres informations à ce sujet.

  • Comment comparer différents tests
  • DIALANG et les tests de langue à l'école
  • DIALANG et les tests de langue sur votre lieu de travail
  • DIALANG et d'autres tests internationaux

DIALANG vous encourage à évaluer d'un point de vue critique quel type d'information peut vous être utile dans l'apprentissage de la langue. Parfois, les résultats de test de langue ou le bilan que vous obtenez de certains tests vous seront utiles pour planifier et guider votre apprentissage, mais ce ne sera pas toujours le cas. Il faut toujours évaluer la crédibilité et l'utilité des résultats obtenus à tel ou tel test.
