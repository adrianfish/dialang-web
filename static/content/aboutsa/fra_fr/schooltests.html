DIALANG et les tests de langue à l'école

Les tests de langue à l'école ou dans d'autres institutions scolaires ont souvent pour but de classer les élèves afin de leur donner une note ou un diplôme sur la base d'un cours suivi ou du contenu de leurs études. Ces tests sont sans doute valables pour les buts qu'ils se donnent, mais pour les apprenants, il est difficile d'en tirer des conclusions sur leur niveau dans l'absolu ou sur la signification des résultats du test en termes de capacité à se servir de la langue. Si votre avis sur vos compétences en langue se base essentiellement sur vos notes et vos tests scolaires, vous aurez peut-être des difficultés à les comparer à une échelle internationale telle que celle du Conseil de l'Europe utilisée par DIALANG. Les contenus et les niveaux (capacités) requis dans les écoles varient selon les pays, mais aussi à l'intérieur d'un même pays ; il est donc difficile de comparer les niveaux exigés au niveau local avec un niveau défini au niveau international.

Vous pouvez étudier de près le contenu et les buts du système d'évaluation DIALANG pour les comparer à vos propres tests scolaires et nationaux, afin de voir à quel point ils se ressemblent ou diffèrent. Ce n'est que par de telles comparaisons que vous pourrez juger si les tests que vous avez effectués dans le passé vous permettent d'évaluer vos aptitudes par rapport aux niveaux utilisés dans le système DIALANG.
