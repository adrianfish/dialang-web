Ύστερα από έντονη εργασία για να νιώσετε ξεκούραστοι και για να βελτιώσετε τη διάθεσή σας, η συνταγή είναι απλή: πάρτε ένα γρήγορο υπνάκο. Πρόσφατες έρευνες δείχνουν ότι δε χρειάζεστε πολύ χρόνο, δε χρειάζεται καν να κοιμηθείτε πραγματικά για να επωφεληθείτε. "Ακόμη και μια σύντομη διακοπή από τις συνηθισμένες σας δραστηριότητες για πέντε δέκα λεπτά μπορεί να έχει θεαματικά αποτελέσματα" υποστηρίζουν οι ερευνητές. Αν δεν είστε από τη φύση σας υπναράδες, σκεφτείτε τι μπορεί να σας κάνει να κοιμηθείτε πιο εύκολα: διαβάστε, δείτε λίγο τηλεόραση και κουλουριαστείτε σε μια ζεστή κουβέρτα.
