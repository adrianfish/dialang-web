Test de niveau

Dans ce test, vous devez décider si les mots existent vraiment ou s'ils sont inventés. Si vous pensez que le mot existe, cliquez sur 'Oui', sinon, cliquez sur 'Non'.<br/><br/>Tous les 'mots' sont des verbes, par exemple, 'parler', 'courir', 'manger', etc.<br/><br/>Si vous changez d'avis, cliquez tout simplement sur l'autre bouton. Le bouton 'Suivant' devient actif dès que vous avez complété le test.<br/><br/>Cliquez dessus pour envoyer vos réponses et passer à l'écran suivant.

Auto-évaluation

Dans cette partie, vous allez évaluer vos propres capacités en déterminant lesquelles des affirmations proposées s'appliquent à vous.

Les affirmations ne sont pas présentées dans un ordre particulier et vous devez les lire toutes très attentivement.

Si vous jugez qu'une affirmation s'applique à vous, cliquez sur 'Oui', sinon cliquez sur 'Non'.

Si vous changez d'avis, cliquez tout simplement sur l'autre bouton.

Le bouton 'Suivant' devient actif dès que vous avez complété le test. Cliquez dessus pour envoyer vos réponses et passer à l'écran suivant.
