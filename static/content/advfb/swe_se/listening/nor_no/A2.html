Följande förslag kan hjälpa dig att avancera till B1:
  • Förvänta dig inte att du direkt skall förstå vad som sägs. Lyssna en gång till, om du kan det, och se om du förstår bättre andra gången.
  • Förbered dig genom att gissa dig till vad det är för tal som du kommer att få höra (utgå från situationen samt från dina förkunskaper om det aktuella ämnet och de människor som är inblandade, dvs dem som du hört eller läst på ditt modersmål).
  • Om det på radio eller TV finns program eller inslag på norska kan du ta för vana att lyssna på vad som sägs och att försöka förstå det väsentliga. Bekymra dig inte om enskilda ord som du missar.
  • Lyssna på sånger på det språk som du håller på att lära dig och läs samtidigt sångtexterna parallellt.
