Följande förslag kan hjälpa dig att avancera till A2:
  • Gå på en språkkurs så att du kan utnyttja en lärares hjälp för att förbättra din strategi för att uppfatta vad som sägs eller följ ett distansinlärningsprogram som kan hjälpa dig att systematisera dina rutiner vid lyssnande.
  • När du lyssnar på spanskt tal kan du försöka använda de namn och de enkla ord som du förstår för att sluta dig till vad resten betyder.
  • Utgå från gester, bilder, intonation eller ljud/musik för få hjälp med att förstå vad den talande menar och vilken attityd denne intar.
  • Var inte rädd för att vid samtal eller när du blir tilltalad begära att man säger om eller formulerar sig på annat sätt. Det gör infödda talare också! Säg helst inte 'No lo entiendo' utan försök hellre med '¿Puede repetir, por favor?', '¿Puede repetir, por favor?' eller till och med '¿Puede hablar más despacio?'. Ju fler tillfällen du får att lyssna, desto bättre kommer du att förstå.
