Die folgenden Vorschläge können Ihnen helfen, den Schritt zur Stufe B1 zu machen:
  • Erweitern Sie die Bandbreite von Texten, die Sie schreiben; schreiben Sie z.B. einen Brief mit demselben Inhalt an verschiedene Leute, z.B. an Freunde, Gäste, Kunden, Arbeitgeber, achten Sie dabei auf den Stil und den erforderlichen Grad an Formalität.
  • Achten Sie auf Wörter, Ausdrücke und Wendungen, die im Spanischen speziell für Argumentationen, Beschreibungen oder zum Ausdruck von Gefühlen und Wünschen verwendet werden; schreiben Sie sie in ein kleines Notizheft, so können Sie die Wendungen immer wieder nachschlagen und leicht finden.
  • Achten Sie darauf, vollständige und gut strukturierte Texte zu schreiben (Textorganisation, Benutzung von Verbindungswörtern); erstellen Sie eine Gliederung, wenn Sie einen längeren Text schreiben.
  • Achten Sie darauf, wie und wann Höflichkeitsformen im Spanischen eingesetzt werden: Unhöflichkeit kann ärgerlicher sein als sprachliche Fehler und kann dazu führen, dass Sie die Ziele, die Sie mit Ihrer Nachricht oder Ihrem Brief erreichen wollen, verfehlen.
  • Versuchen Sie, jeden Tag etwas auf Spanisch zu schreiben, führen Sie z.B. Tagebuch.
