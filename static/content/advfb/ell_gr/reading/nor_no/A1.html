Οι παρακάτω υποδείξεις μπορεί να σας βοηθήσουν να σημειώσετε πρόοδο προς το Α2:
  • Καλό θα είναι να παρακολουθήσετε γλωσσικά μαθήματα για να μάθετε περισσότερη γραμματική στο επίπεδο δομής των προτάσεων και νορβηγικό λεξιλόγιο, δίνοντας προσοχή στο σχηματισμό των προτάσεων και στη σύνδεση των κύριων προτάσεων με τις δευτερεύουσες.
  • Προσπαθείτε ενεργά να χρησιμοποιείτε τα νορβηγικά που ξέρετε για να διαβάζετε ετικέτες σε πακέτα, επιγραφές και καταλόγους φαγητών, επαληθεύοντας όπου είναι δυνατό μέσα από τις ανάλογες μεταφράσεις στη δική σας γλώσσα.
  • Χρησιμοποιήστε τη διάταξη του κειμένου ή οποιαδήποτε εικονογράφηση ή αναπαράσταση καθώς και τα κύρια ονόματα ή απλές λέξεις μέσα στο κείμενο για να υποθέσετε τη σημασία λέξεων ή φράσεων που δεν καταλαβαίνετε.
  • Μάθετε να χρησιμοποιείτε ένα δίγλωσσο λεξικό ή ένα απλό μονόγλωσσο. Τα λεξικά προσφέρουν μεγάλη βοήθεια, αν χρησιμοποιούνται σωστά.
  • Διαβάζετε εύκολα, απλοποιημένα λογοτεχνικά κείμενα, αν βέβαια υπάρχουν. Αυτά συνήθως ετοιμάζονται και απλοποιούνται προσεκτικά, ώστε να μπορείτε να τα καταλαβαίνετε. Ψάξτε σε καλά βιβλιοπωλεία ή βιβλιοθήκες, ιδιαίτερα σε χώρες στις οποίες μιλιούνται τα νορβηγικά.
  • Να εκμεταλλεύεστε οποιαδήποτε ευκαιρία σας δίνεται για να διαβάζετε, να ακούτε ή να χρησιμοποιείτε νορβηγικά σ' αυτό το επίπεδο. Οποιαδήποτε έκθεση στα νορβηγικά είτε σε κατανόηση και παραγωγή γραπτού ή προφορικού λόγου πιθανότατα να σας βοηθήσει να βελτιώσετε την ικανότητά σας να κατανοείτε το γραπτό λόγο.
