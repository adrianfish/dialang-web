Οι παρακάτω υποδείξεις μπορεί να σας βοηθήσουν να σημειώσετε πρόοδο προς το B2:
  • Προσπαθείτε να διαβάζετε καταλαβαίνοντας ενεργά ευρύτερη ποικιλία κειμένων όπως: πιο δύσκολα κείμενα του αντικειμένου σας, εξειδικευμένα κείμενα άλλων τομέων, δελτία ειδήσεων στα ιρλανδικά τα οποία εκφράζουν συγκεκριμένες απόψεις ή σημεία διαφωνίας.
  • Αναζητήστε τους λόγους για τους οποίους διαβάζετε ένα κείμενο (για διασκέδαση, για γενικές ή συγκεκριμένες πληροφορίες, για να ολοκληρώσετε κάποια εργασία) και αναλύστε πώς μπορείτε να προσαρμόσετε το δικό σας τρόπο ανάγνωσης (γρήγορη ανάγνωση ενός κειμένου για να βρείτε την ουσία, γρήγορη ανάγνωση για να ανακαλύψετε μια συγκεκριμένη πληροφορία, διεξοδική ανάγνωση για να βρείτε τις λεπτομέρειες μιας συγκεκριμένης διαδικασίας.
  • Αρχίστε να διαβάζετε μη απλοποιημένα αλλά όχι πολύ δύσκολα μυθιστορήματα. Αρχίστε να διαβάζετε ένα μυθιστόρημα κάποιου συγγραφέα που πραγματικά σας αρέσει ή κάποιο που ήδη έχετε διαβάσει στη γλώσσα σας. Ο δάσκαλός σας ή φίλοι σας που μιλούν τη γλώσσα μπορούν να σας βοηθήσουν στην επιλογή σας.
  • Πότε πότε διαβάζετε κείμενα λεπτομερώς και σε βάθος, ίσως με τη βοήθεια ενός λεξικού. Πιθανόν ήδη να τα καταφέρνετε αρκετά καλά με καθημερινά κείμενα σε αυτό το επίπεδο, αλλά θα απαιτηθεί συνειδητή προσπάθεια από μέρους σας για να προχωρήσετε στο επόμενο επίπεδο.
