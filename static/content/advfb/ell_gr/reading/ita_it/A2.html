Οι παρακάτω υποδείξεις μπορεί να σας βοηθήσουν να σημειώσετε πρόοδο προς το B1:
  • Διευρύνετε την ποικιλία και τον τύπο των κειμένων που προσπαθείτε να διαβάζετε ενεργά. Θα ήταν καλό να ρίχνετε μια ματιά σε εφημερίδες και περιοδικά στα ιταλικά και να δουλεύετε ένα ή δύο από τα πιο δύσκολα άρθρα που σας ενδιαφέρουν, με τη βοήθεια λεξικού, αν αυτό είναι απαραίτητο.
  • Δίνετε ιδιαίτερη προσοχή σε συγκεκριμένες εκφράσεις, φράσεις και λέξεις στα ιταλικά που χρησιμοποιούνται για επιχειρηματολογία, περιγραφές και για την έκφραση συναισθήμάτων και επιθυμιών.
  • Προσπαθείτε να καταλαβαίνετε τις λεπτομέρειες παρά την κύρια ιδέα συγκεκριμένων κειμένων. Να έχετε ένα σημειωματάριο για να γράφετε τις εκφράσεις που νομίζετε ότι είναι χρήσιμες και αξίζει να τις θυμάστε.
  • Βεβαιωθείτε ότι διαβάζετε πολύ στα ιταλικά. Αν και μπορείτε ήδη να καταλαβαίνετε πολλά από αυτά που διαβάζετε, θα πρέπει να αναπτύξετε ενεργά τις ικανότητές σας στην κατανόηση γραπτού λόγου. Βεβαιωθείτε ότι διαβάζετε πολλά κείμενα και περιστασιακά πιο δύσκολα και σε βάθος.
