Οι παρακάτω υποδείξεις μπορεί να σας βοηθήσουν να σημειώσετε πρόοδο προς το Α2:
  • Καλό θα είναι να παρακολουθήσετε γλωσσικά μαθήματα, ώστε να ωφεληθείτε από τη βοήθεια ενός δασκάλου, προκειμένου να βελτιώσετε τις στρατηγικές σας στην κατανόηση προφορικού λόγου ή να λάβετε μέρος σε ένα πρόγραμμα εξ αποστάσεως εκμάθησης της γλώσσας. Αυτό μπορεί να σας βοηθήσει να εξασκηθείτε συστηματικά στην κατανόηση προφορικού λόγου.
  • Όταν ακούτε Φινλανδικά, χρησιμοποιείτε τα ονόματα και τις απλές λέξεις που καταλαβαίνετε, για να υποθέσετε τη σημασία όσων δεν καταλαβαίνετε.
  • Χρησιμοποιήστε οπτικά βοηθήματα εικόνων, επιτονισμού ή ήχου / μουσικής, για να μπορέσετε να καταλάβετε τι εννοεί ο ομιλητής ή ποια είναι η άποψή του.
  • Σε συνομιλίες ή όταν κάποιος απευθύνεται σ' εσάς, μη φοβάστε να του/της ζητήσετε επανάληψη ή επαναδιατύπωση. Αυτό το κάνουν ακόμα και οι φυσικοί ομιλητές! Αποφύγετε τα 'En ymmärrä.' και αναφερθείτε σε 'Anteeksi, voisitteko toistaa?', 'Voisitteko sanoa uudestaan.', ακόμα και 'Voisitteko puhua hitaammin?'. Όσο περισσότερες ευκαιρίες έχετε να ακούσετε κάτι, τόσο καλύτερα θα το καταλάβετε.
