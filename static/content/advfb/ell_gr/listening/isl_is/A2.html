Οι παρακάτω υποδείξεις μπορεί να σας βοηθήσουν να σημειώσετε πρόοδο προς το B1:
  • Μην περιμένετε να καταλαβαίνετε πράγματα ευθύς εξαρχής. Ακούστε ξανά, αν μπορείτε, και δείτε αν καταλαβαίνετε περισσότερα τότε.
  • Να προετοιμάζεστε, προβλέποντας τον τύπο τής ομιλίας που θα ακούσετε (χρησιμοποιείτε συμφραζόμενα, προηγούμενη γνώση για το θέμα και τους ανθρώπους, που έχετε ακούσει ή διαβάσει στην πρώτη σας γλώσσα).
  • Αν υπάρχει κάτι στο ραδιόφωνο ή στην τηλεόραση στα Ισλανδικά, μην παραλείπετε να το ακούτε και να προσπαθείτε να καταλαβαίνετε το κύριο νόημα, χωρίς να σας απασχολούν οι μεμονωμένες λέξεις που μπορεί να μην καταλαβαίνετε.
  • Ακούστε τραγούδια στη γλώσσα που μαθαίνετε και παρακολουθήστε τα λόγια τους.
