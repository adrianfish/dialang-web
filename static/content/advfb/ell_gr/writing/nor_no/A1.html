Οι παρακάτω υποδείξεις μπορεί να σας βοηθήσουν να σημειώσετε πρόοδο προς το Α2:
  • Καλό θα είναι να παρακολουθήσετε γλωσσικά μαθήματα για να μάθετε περισσότερη γραμματική στο επίπεδο δομής των προτάσεων και νορβηγικό λεξιλόγιο, δίνοντας προσοχή στο σχηματισμό των προτάσεων και στη σύνδεση των κύριων προτάσεων με τις δευτερεύουσες.
  • Μάθετε τις βασικές συμβάσεις για τη συγγραφή επιστολών στα νορβηγικά, για παράδειγμα, πώς να αρχίζετε και πώς να τελειώνετε μια επιστολή σε ένα φίλο ή σε κάποιον που δε γνωρίζετε.
  • Εκμεταλλευτείτε όποιες ευκαιρίες μπορείτε για να διαβάσετε, να ακούσετε ή να χρησιμοποιήσετε νορβηγικά: σε αυτό το επίπεδο οποιαδήποτε έκθεση στα νορβηγικά πιθανότατα θα σας βοηθήσει να βελτιώσετε την ικανότητά σας στην παραγωγή γραπτού λόγου.
  • Έχετε ένα σημειωματάριο στο οποίο να γράφετε τις νέες λέξεις και τις νέες δομές για να τις μαθαίνετε και για να βελτιώνετε την ορθογραφία σας.
  • Προσπαθήστε να μάθετε πώς σχηματίζονται συνήθως οι νέες λέξεις στα νορβηγικά.
