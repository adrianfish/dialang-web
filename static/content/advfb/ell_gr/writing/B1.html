Ποιους τύπους κειμένων μπορώ να γράψω Συνήθως μικρά, απλά κομμάτια γραπτού λόγου. Για παράδειγμα, απλά προσωπικά γράμματα, κάρτες, μηνύματα, έντυπα. Μπορώ να γράψω ένα συνεχές κατανοητό και συνεκτικό κείμενο. Μπορώ να γράψω ποικίλα διαφορετικά κείμενα.
Τι μπορώ να γράψω Αυτά τα κείμενα συνήθως περιγράφουν άμεσες ανάγκες, προσωπικά γεγονότα, γνωστά μέρη, ενασχολήσεις, εργασία κ.τ.λ..

Αυτά τα κείμενα συνήθως αποτελούνται από σύντομες κύριες προτάσεις.

Μπορώ να χρησιμοποιήσω τους πιο συχνούς συνδέσμους (και, αλλά, διότι) για να συνδέσω προτάσεις, με σκοπό να αφηγηθώ μια ιστορία ή να περιγράψω κάτι, όπως έναν κατάλογο από στοιχεία.
 Μπορώ να μεταφέρω απλές πληροφορίες σε φίλους, σε υπαλλήλους κ.ά., που εμφανίζονται στην καθημερινή ζωή.

Μπορώ να αποδώσω σωστά και κατανοητά απλές πληροφορίες.

Μπορώ να μεταδώσω ειδήσεις, να εκφράσω σκέψεις για αφηρημένα ή πολιτιστικά θέματα όπως κινηματογραφικές ταινίες, μουσική κ.ά..

Μπορώ να περιγράφω εμπειρίες, συναισθήματα και γεγονότα, με κάποιες λεπτομέρειες.
 Μπορώ να εκφράσω αποτελεσματικά ειδήσεις και απόψεις και να τις συσχετίσω μ' αυτές που εκφράζουν οι άλλοι.

Μπορώ να χρησιμοποιήσω ποικιλία συνδετικών λέξεων για να επισημάνω καθαρά τις σχέσεις μεταξύ ιδεών.

Η ορθογραφία και στίξη χρησιμοποιούνται με σχετική ακρίβεια.
Όροι και προϋποθέσεις Μόνο σε οικεία και συνηθισμένα θέματα.

Η σύνταξη συνεχούς συνοχικού κειμένου είναι δύσκολη.		 Η εμβέλεια των κειμένων μπορεί να περιορίζεται στα πιο οικεία και συνηθισμένα, όπως είναι οι περιγραφές αντικειμένων ή η γραπτή παρουσίαση μιας ακολουθίας πράξεων. Αλλά η επιχειρηματολογία και θέματα με αντιθέσεις, για παράδειγμα, είναι δύσκολα. Είναι συνήθως δύσκολο το να αποδίδει κάποιος λεπτές αποχρώσεις, όταν εκφράζει την άποψή του ή όταν μιλά για συναισθήματα και εμπειρίες.
